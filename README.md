# DrawPoker

## 概要

UnityとPhoton PUN2を用いて制作した、オンライン対戦型のドローポーカーゲームです。

複数人が同じルームに参加し、5枚の手札配布、カード交換、参加/フォールド選択、役判定、勝敗判定までをオンラインで行えるように実装しました。

本作品は、カードゲーム中の判断や手札の強さとプレイヤー状態の関係を調べるための予備実験用システムとして制作しました。

## 特徴

- Unityによるドローポーカー実装
- Photon PUN2を用いたオンラインマルチプレイ通信
- 各プレイヤーへの5枚の手札配布
- カード選択による交換処理
- 交換を行わないスキップ処理
- 参加 / フォールド選択
- フォールド回数の管理
- ポイント制による勝敗管理
- ポーカー役の判定と勝者決定
- ラウンド数と勝利ポイントの管理
- ゲームログの保存
- 実験用の注視フェーズ表示

## 主な実装

- MultiplayerGameManager.cs
  - ゲーム進行、カード配布、カード交換、参加/フォールド処理、ポイント管理、勝敗判定、ログ保存を担当

- PokerUIManager.cs
  - カード交換ボタン、スキップボタン、参加/フォールドボタン、ポイント表示、ラウンド表示、勝者表示、ログ表示を担当

- PokerHandEvaluator.cs
  - 5枚の手札からポーカー役を判定し、役の強さとキッカーを比較して勝者を決定

- PhotonLauncher.cs
  - Photonへの接続、ルーム参加、ゲーム開始処理を担当

- Card.cs
  - カードの数字、マーク、表向き・裏向き表示、クリック選択、交換対象の選択状態を担当

## ゲームの流れ

1. Photonのルームに参加
2. デッキを生成・シャッフル
3. 各プレイヤーに5枚の手札を配布
4. カード交換フェーズに移行
5. プレイヤーが交換したいカードを選択
6. 選択したカードを新しいカードに交換、または交換をスキップ
7. 参加 / フォールドを選択
8. 参加者の手札で役判定
9. 勝者にポイントを加算
10. 次ラウンドへ進行
11. 最大ラウンドまたは勝利ポイント到達で最終勝者を表示

## 実験用機能

本ゲームには、予備実験で利用するための注視フェーズ表示を実装しています。

カード交換フェーズや参加判断フェーズに合わせて、画面上の注視点表示を切り替えることで、プレイヤーが特定のタイミングで画面を注視できるようにしています。

なお、本リポジトリのプログラムには、脳波の取得・解析・共有処理は含まれていません。

## 工夫した点

ドローポーカーでは、配られた5枚の手札からどのカードを交換するかを選ぶ必要があります。

そのため、カードごとにクリック選択状態を持たせ、交換フェーズ中のみカードを選択できるようにしました。

また、オンライン対戦で各プレイヤーの手札、交換状況、参加/フォールド状態、ポイント、勝敗結果が一致するように、Photon RPCを用いて状態を同期しています。

実験用途として、ゲーム進行ログを保存し、ラウンドごとの手札、交換枚数、役、勝敗を後から分析しやすいようにしました。
